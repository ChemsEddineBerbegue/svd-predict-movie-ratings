Singular Value Decomposition Movie Recommendation Algorithm

10M of Original Ratings - Sparse Matrix

~ 3.5 GB RAM

Compute Feature Vectors that will predict ratings for all values

error = (A' - Aa) * lrate

Error is the difference between the original rating (A') and product of vectors (Aa), times learning rate (lrate).

uF[u] += error * mF[m] mF[m] += error * uF[u]

Assign initial value to the vector, and train each vector in a loop

Multiply all vectors accordingly to get predictions

Substract prediction matrix from the original ratings to get the error matrix. Train new set of vectors on the error matrix.

Get a set of vectors, each vector represents a feature.
The sum of the products of all vectors produces an accurate prediction

Each feature has an effect on users' ratings - from highest (first feature) to lowest (last feature)
